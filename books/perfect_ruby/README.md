# ruby

rubyは全てがオブジェクトである.そのため、数値や真偽値、nilなどのプリミティブ型のような方もオブジェクトであるため、メソッドを読み出すことが可能.
```ruby
p 1.to_s     # => "1"
p true.to_s  # => "true"
p false.to_s # => "false"
p nil.to_s   # => ""
```
