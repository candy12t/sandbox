# password_generator

## Usage
```
Usage:
  password_generator generate

Options:
  -l, [--length=N]               # word length
                                 # Default: 18
  -n, [--number], [--no-number]  # include a number
                                 # Default: true
  -s, [--symbol], [--no-symbol]  # include a symbol

Generate password.
```

ex
```
password_generator generate --length=24 --symbol
doP,Y!V7E`Xp9T+xNEP/aT%t`
```
